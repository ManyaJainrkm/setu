# Setu — AI Concept Prototype

**Live demo:** https://95725f63e1f1d569a51bb5a5c2062125.ctonew.app/

Setu ("bridge") is a concept prototype exploring specialized AI conversational 
support agents — each modeled with a distinct persona, background, and area 
of focus, rather than a single general-purpose assistant.

## What's in this repo

This repo contains the design and product artifacts for the concept:

- `personas.md` — specifications for 7 distinct agent personas
- `design/personas/` — visual identity for each persona
- `design/chat-mockup.html` — interface mockup for the conversation flow
- `design/style-guide.md` — visual design system

The live demo (linked above) reflects the working interface; this repo 
documents the product thinking behind it.

## Process

Built end-to-end using AI coding agents (cto.new), with the product 
architecture, persona design, and session flow direction done by me. 

## Status

Early-stage prototype / concept exploration — not a live product.
